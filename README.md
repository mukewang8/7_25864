<h3>课程介绍：</h3>
<p>Java目前作为游戏服务器开发语言已经很是普遍。但是大多数是作为页游或手游的服务器，而端游戏一般选择C++，因为端游对服务器的性能要求相对比较高一些。两种语言各有利弊。C++效率高，但是掌握度难些。没有Java易于掌握。而目前对于追求快速上线率的页游和手游来说，Java成了一个不错的选择。</p>
